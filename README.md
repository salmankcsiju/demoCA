# Casa Amora - Boutique Atelier & ERP

A high-end, premium web platform for a boutique tailoring atelier, featuring a cinematic customer experience and a robust Staff ERP/CRM portal.

## 🚀 Presentation Features

### Customer Experience (Registry)
- **High-End Customizer**: Bespoke tailoring workflow with WhatsApp integration.
- **Visual Storytelling**: Dynamic hero sections and artisan diaries.
- **Membership Registry**: Personal profiles with order history and wishlists.

### Staff Portal (Atelier Mission Control)
- **Revenue Dashboard**: Real-time business metrics.
- **Inventory ERP**: Full CRUD management of Products, Categories, and Fabrics.
- **CRM Suite**: Detailed customer dossiers with lifetime value tracking.

## 🛠 Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Django, Django REST Framework, SQLite (Local Demo).
- **Security**: JWT Authentication with role-based access.

## 🏁 Quick Start (Local Demo)

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python manage.py migrate`
4. `python manage.py runserver`

### Frontend
1. `cd website`
2. `npm install`
3. `npm run dev`
